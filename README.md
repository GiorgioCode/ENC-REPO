# 01 DEFINICIONES ENC IHO

-   ENC - ELECTRONIC NAUTIC CHART
-   IHO - INTERNATIONAL HYDROGRAPHIC ORGANIZATION

# 02 DIFERENCIA ENTRE CARTAS NAUTICAS PAPEL, RASTER Y ENC IHO

-   CARTA PAPEL SE MANEJA PENSANDO EN EL PAPEL Y EN EL TAMAÑO Y CANTIDAD DE CARTAS QUE SE PUEDEN GUARDAR
-   CARTAS RASTER (A GRANDES RAZGOS, ESCANEO CARTA PAPEL), DIGITALIZAN EL SISTEMA DE PAPEL. SOLO TIENEN GEOREFERENCIACION Y LOS OBJETOS DE INFORMACION, AL ESTAR INCRUSTADOS EN LA IMAGEN, NO SON INTERACTIVOS Y LEGIBLES DIGITALMENTE
    -   PROBLEMAS:
        -   PIXELADO
        -   INCONSISTENCIA GRAFICA - SOLAPAMIENTO
        -   PROBLEMAS O IMPOSIBILIDAD DE CORRECCIONES
-   CARTAS RASTER+ EXISTEN VERSIONES DE CARTAS RASTER QUE AGREGAN UNA CAPA DE DATOS CON OBJETOS DE INFORMACION
-   ENC (ELECTRONIC NAUTIC CHART) ES UN SISTEMA COMPLETAMENTE PENSADO PARA SER UTILIZADO EN FORMA ELECTRONICA. LA ORGANIZACION DEL CONTENIDO SE DIVIDE EN CELDAS QUE YA NO SON NECESARIAMENTE RECTANGULARES Y NO TIENEN LIMITACIONES EN CANTIDAD O TAMAÑO.
-   POSEEN VARIAS CAPAS DE DATOS Y BASICAMENTE SON UNA BASE DE DATOS. LOS DATOS GRAFICOS SE ENCUENTRAN EN EL VISOR O MOTOR DE VISUALIZACION. EXISTE UN MOTOR GRAFICO OFICIAL DE IHO
